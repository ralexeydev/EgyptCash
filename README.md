# EgyptCash

EgyptCash is a Unity-based slot game inspired by the mysteries and treasures of ancient Egypt.

The project features classic slot gameplay combined with Egyptian-themed symbols, animated reels, reward mechanics, and configurable gameplay systems.

## Features

- 2D slot gameplay
- Egyptian-themed visuals and symbols
- Wild and multiplier mechanics
- Deterministic result generation
- XML-driven configuration
- Modular architecture
- Mobile-ready structure

## Tech Stack

- Unity
- C#
- XML configuration
